### Key fixes applied during implementation

- `--border-subtle`, `--surface`, `--accent` aliases added to both `:root` and `[data-theme="dark"]` (translate page used them)
- `@keyframes spin`, `bounce-dot`, `tts-toast-in/out` moved from inline blocks to `styles.css`
- `filter-chip.active` CSS rule added; JS updated to use `classList.toggle` instead of inline `style`
- `d.collections` unpacked in stats fetch on `index.html` to match actual API response shape `{ collections: {...}, total_documents: N }`
- `.card` removed from `.card-scene` wrapper on teach page (would have broken 3D flip geometry)
- `border-radius:50` → `border-radius:50%` typo fixed in translate.html
- `btn-primary.is-loading` rule added to styles.css (suppresses hover lift during active request)
- PWA meta tags restored to all pages
- Duplicate `@keyframes` deduplicated
